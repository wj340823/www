Target tool

The Target tool continuously tracks the distance and direction to a specific target.

To use the Target tool:

1.     Enter the following information:

Field

Description

Target Set Mode

Select either:

§   Numeric Entry

§   Manual Selection

Target X, Target Y, Target H

If you selected Numeric Entry, enter the target's X, Y and Height coordinates.

Target Mark scale

Enter the Target icon maximum scale (pixels per meter).

Display Units

Select the display units to be used for distance/height information display.

Show Direction Arrow

Select Yes if you want to display the direction-to-target arrow.

Show MGRS

Select Yes if you want to display the coordinates in MGRS system.

2.     If you are using Manual Selection as the Target-Set-Mode, click on the terrain or on any object in the 3D Window.

3.     The tool displays the Camera Location, Cursor Location, Target Location and Distance from Target values according to the selected units.

4.     To reset this tool, click the Reset button.
