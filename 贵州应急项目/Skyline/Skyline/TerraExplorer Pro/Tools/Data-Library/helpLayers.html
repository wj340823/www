Layers Library tool

The Layers Library contains feature layers that you can add to your project. The layers are stored in Fly projects you can add to your project.

This group has two sub-groups:

§         World Layers: A set of feature layers that includes elements around the globe.

§         US Layers: A set of feature layers that includes elements in the United States of America.

Note:    The feature layers in the Data Library are set to work with projects using Globe terrain models with Lat-Long projections. You can load the feature layers to local projection terrain only if the project contains the projection of the terrain. See: “Projection” setting in the “Setting the Project” chapter for further information.
