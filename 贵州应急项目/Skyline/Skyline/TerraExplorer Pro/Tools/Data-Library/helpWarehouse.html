3D Warehouse tool

Sketchup 3D Warehouse is an online repository of 3D models.

To use Sketchup Warehouse:

1.     Enter the model to search for and click Search. The Sketchup 3D Warehouse opens displaying a collection of relevant thumbnail images.

2.     Click the thumbnail image of the model you want to download, click Download, and then from the drop-down list of formats, select either: KMZ from SketchUp or Collada File (.zip).

3.     Save the model to your local or network drive.

4.     If you downloaded a model in KMZ format, load the model into your project by using TerraExplorer’s Load KML/KMZ command.

5.     If you downloaded a Collada (DAE) model, open the .zip file, and extract its contents to a folder. Then load the .dae file into your project by using TerraExplorer’s Load 3D Model command.
