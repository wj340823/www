Timespan tool

Timespan tags restrict the visibility of data to a specific period of time. Timespans are usually used for data sets that appear in multiple locations (e.g., Placemarks moving along a path). The data associated with a timespan is visible only when the time slider is showing the time interval defined in the timespan tag.

The timespan for an individual object can be set in its property sheet. Similarly, the timespan for a group can be set from the group property sheet, when you want all objects in the group and all subgroup objects to have the same timespan. The Timespan Tool is necessary when you want to evenly split a timespan, either between a group’s subgroups or between all the objects in all of a group’s subgroups.

Using the Timespan Tags Tool

1.     In the Project Tree, select the group.

2.     Click the Enter Start Date and Time icon and enter a start date and time.

3.     Click the Enter End Date and Time icon and enter an end date and time.

4.     The defined time interval is split evenly among all the objects and subgroups in the group, with each object displaying after the one that preceded it on the Project Tree, for an equal fraction of the total time.

5.     If the group contains subgroups, in the Subgroup Objects field, define how the objects within each subgroup should display:

·          Consecutive - The objects within each subgroup display successively, each for a fraction of the total timespan.

·          Simultaneous - Each subgroup should function like a unified object, with all the objects within each subgroup displaying simultaneously during the subgroup’s segment of the total timespan.

6.     Click Apply to Group.

7.     If you want to display a table summarizing the date, hour, and duration that each group object will display, click Show result.

If you want to clear the timespan set for a particular group:

1.     On the Tools tab, in the Time group, click Timespan Tags.

2.     In the Project Tree, select the required group.

3.     In the Timespan Tags dialog box, click Clean Time.
