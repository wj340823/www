Indoor Views and Navigation tool

The Indoor Views tool creates indoor locations and entrance icon for 3D models. Clicking the indoor entrance icon opens the Indoor Navigation popup that enable you to jump between the different locations and navigate freely inside the building. The tool also supports underground views.

Creating Indoor Locations

To create Indoor Locations:

1.     Click the Select Building button.

2.     Click the building you wish to select. This building should be a 3D model (.x, .xpl, .dae or .flt) and include interior modeling of the different floors and rooms. The Indoor Editor - Step 2 dialog box opens.

3.     Place the camera in the first location in the building, type the name of the location and click Add.

4.     Repeat step 4 for each location you want to create in the building.

5.     Click Finish. You can create several locations in different floors and rooms.

6.     If you want to delete a location, click the Delete  icon next to the required location.

7.     If you want to jump to a specific location, click the View  icon next to the location.

Activating the Indoor Navigation Mode

To activate the indoor navigation mode:

1.     Click the Enter icon above the building. The Indoor Navigation tool is displayed in a popup window. This tool changes the navigation modes to enable easy navigation inside the building.

 

Indoor Mode Tool

2.     If you want to jump to a specific location, click the location’s name.

3.     If you want to turn Collision Detection on, select the On option button.

4.     If you want to exit from the indoor navigation mode , click the Exit Indoor Mode link or the X button.

Note:    If you want to attach the indoor navigation tool to the project folder, publish the project as a local kit or as an Internet kit.

Note:    You cannot create indoor locations for buildings under streaming layers. You should place the building manually or load them with a non-streamed layer.
