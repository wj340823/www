Shadow Analysis tool

The Shadow Analysis tool calculates the shadow cast from buildings and 3D objects in a given radius based on the sun’s position. The sun is positioned according to the time, date and time zone you set using the Date and Time controls.

The shadows dynamically update when the system date and time is changed.

Creating a Shadow

To create a shadow:

1.     In the 3D Window, set the time, date and time zone using the Date and Time controls.

2.     Navigate the camera to the desired position. A shadow is cast on buildings and 3D objects positioned around the center of the 3D Window.

3.     In the Shadow Analysis dialog box, type the radius from the center point for which a shadow should be created.

Note: It is recommended to limit the shadow region only for the area of interest. Casted shadow models may be heavy in highly detailed urban environments.

4.     If you want to display the shadow in the 3D Window, click the Show Radius link.

5.     Click Create Shadow to create the shadow.

6.     If you want to remove the shadow, click Clear. If you want to remove the previous shadow and create a new one, click Create Shadow.
