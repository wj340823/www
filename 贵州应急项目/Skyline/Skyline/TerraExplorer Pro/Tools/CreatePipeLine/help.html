Create Pipe Lines tool

This tool creates 3D pipe lines along a user-defined path.

The output of a build pipe line operation is either a group of TerraExplorer objects or two Points layers, one for the pipes and the second for the connectors, which can be saved to local data sources (Shape file format) and loaded as a Streaming or Entire layers.

To create pipelines:

1.     Select the Create as option:

Group

Create a group with objects.

Layer (Entire)

Create feature layer under the application AppData.

Layer (Streaming)

Create streaming feature layer under the application AppData.

2.     Enter the following parameters:

Parameter

Description

Pipe Radius

Type the desired radius.

Pipe Color

Select the required color.

Visibility Distance

Set the visibility distance of the objects.

3.     If you want to create pipe lines along all lines in a selected group or layer, click Selected Group, and then select the required group or layer from the Project Tree.

4.     If you want to create pipe lines along a specific line, select Follow Line. In the 3D Window, click the left mouse button to place the line waypoints, and then right-click to complete the line.

5.     If you want to create pipe lines along selected objects or features in the project, copy the required objects to the clipboard and click From Clipboard

6.     If you want to edit additional parameters, right-click the Pipe Line group in the Project Tree and select Properties.
