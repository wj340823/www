Build Power Line tool

This tool creates a power line by placing power poles along a user-defined path and connecting them with power lines.

The output of a build power line operation is either a group of TerraExplorer objects or a Points layer that can be saved to a local data source (Shape file format) and loaded as a Streaming or Entire layer.

To use the Build Power Lines tool:

1.     Select the Create as option:

Group

Create a group with objects.

Layer (Entire)

Create feature layer under the application AppData.

Layer (Streaming)

Create streaming feature layer under the application AppData.

 

2.     Enter the following parameters:

Parameter

Description

Method

Select the required creation method:

§   Pole every waypoint - To add a single pole on every left-click on the terrain.

§   Fixed distance between poles - To add several poles between clicks with a fixed distance between each pole.

Pole Type

§   Pole Type A creates a power line with two connecting wires.

§   Pole Type B and Pole Type C create a power line with three connecting wires.

Distance between poles

This field is enabled when the Fixed distance between poles method is selected.

Segments in sections

The number of segments for connected power lines between two poles.

Maximum Line Depth

The height difference of the connected power lines between the highest and lowest points.

Visibility Distance

 

Set the visibility distance of the objects.

 

Line Color

The color of the drawn power line.

3.     If you want to build power lines along all lines in a selected group or layer, click Selected Group, and then select the required group or layer from the Project Tree.

4.     If you want to build power lines along a specific line, select Follow Line. In the 3D Window, click the left mouse button to place the line waypoints, and then right-click to complete the line.

5.     If you want to build power lines along selected objects or features in the project, copy the required objects to the clipboard and click From Clipboard.

6.     If you want to edit additional parameters, right-click the Power Line group in the Project Tree and select Properties.
