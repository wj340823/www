Image Comparison tool

The Image Comparison tool helps you detect changes between two aerial images of the same location.

 The Image Comparison provides two useful tools allowing you easy manual detection of changes between two aerial images or between two project scenarios.

Using the Imagery Layer Swipe & Fade

 The Imagery Layer Swipe & Fade tool let you compare the 3D view with and without an Imagery layer

 1. Zoom to an area with visible Imagery Layer

 2. Click the Swipe & Fade button

 The Imagery Layer Swipe & Fade tool opens with the following controls:

 1. Transparency Slider – change the transparency slider to control the imagery layer opacity.

 2. Clicking the Auto button automatically change the layer opacity value between 0-100%.

 3. Clip Slider – change the Clip slider to reveal the base terrain under the Imagery Layer.

 4. Clicking the Auto button automatically change the clipping value between 0-100%.

Using the 3D View Comparison

 The 3D View Comparison tool let you compare snapshots of two project scenarios.

 1. Fly to a location of you desire

 2. click Capture A

 3. Turn on the different layers in the Project Tree

 4. click Capture B

 5. click Compare

 The 3D View Comparison tool opens with the following controls:

 1. Transparency Slider – change the transparency slider to control the Scenario B snapshot transparency and slowly reveal the underneath Scenario A snapshot.

 2. Clip Slider – change the Clip slider to reveal part of the underneath Scenario A snapshot.
